### DB
USER
```
  id VARCHAR(24) PRIMARY KEY
  pw VARCHAR(16)
  img VARCHAR(100)
  name VARCHAR(16)
```
POST
```
  userId VARCHAR(24)
  postTime DATETIME
  postNum INT(10) PRIMARY KEY AUTO_INCREMENT
  likeNum INT(5)
  content TEXT
```
